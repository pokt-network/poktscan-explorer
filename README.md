# Requirements

- Use pnpm
- Node version recommended: ^18.19.0
- You must set the GRAPHQL_API_URL environment variable to the .env file.

## How to run

```bash
pnpm install
pnpm dev
```

## How to generate the GraphQL types

```bash
pnpm codegen
```

## How to build

```bash
pnpm build
```

## How to start production

```bash
pnpm start
```
