# Requirements

- Use pnpm
- Node version recommended: ^18.19.0
- You must set the following environment variables to the .env file:
  - GRAPHQL_API_URL
  - RPC_BASE_URL

## How to run

```bash
pnpm install
pnpm dev
```

## How to generate the GraphQL types

```bash
pnpm codegen
```

## How to build

```bash
pnpm build
```

## How to start production

```bash
pnpm start
```
